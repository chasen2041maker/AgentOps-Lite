# GroundGuard v0.2.1

GroundGuard v0.2.1 focuses on the first 30 seconds of adoption: install it,
run a demo, run a benchmark, then wire it into CI with a config file.

## Highlights

- `groundguard-demo` runs a packaged before/after fact-gate demo after install.
- `groundguard-benchmark` prints deterministic smoke metrics after install.
- `groundguard.example.yml` gives teams a starting point for CI/eval config.
- MkDocs pages, GitHub Pages workflow, publishing workflow, security policy,
  code of conduct, and issue template make the repository easier to evaluate.
- Runnable promptfoo and DeepEval examples show how the assertion schema fits
  into common eval workflows.

## Install

```bash
python -m pip install "git+https://github.com/chasen2041maker/GroundGuard.git@v0.2.1"
groundguard-demo
groundguard-benchmark
```
