# GroundGuard S0-S2 Core Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the first runnable Python package slice: project skeleton, pytest setup, core models, Ledger, and tool_call.

**Architecture:** Keep v1 local-first and deterministic. `core` modules pass plain dataclasses between boundaries; `Ledger` owns in-memory fact storage and JSONL persistence; `tool_call` is the only v1 helper for explicit fact registration from tool results.

**Tech Stack:** Python 3.11+, dataclasses, Decimal, context managers, JSONL, pytest.
